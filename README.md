# ✨ Sushma Sharma's Personal Portfolio & Blog 🚀

Welcome to the official repository of my personal portfolio and developer blog website. This is a premium, modern, and highly interactive digital space designed to showcase my journey, projects, skills, and insights.

---

## 🎨 Design Theme & Aesthetics

This project is built with a premium **Neo-Dark Cyberpunk Glassmorphism** design:
* 🌌 **Background**: Deep cybernetic dark overlay with grid lines (`grid-overlay`).
* 🧪 **Accents**: Neon Cyan (`#00E5FF`) and Purple (`#7C3AED`) glow effects.
* ✨ **Interactivity**: Smooth mouse/scroll responsive glows, interactive typewriter animations, and custom page transitions.
* 🌟 **Polished Feel**: Micro-animations powered by Framer Motion, GSAP, and smooth kinetic scrolling with Lenis.

---

## ⚡ Key Features

This portfolio is packed with rich features:

* 🌐 **Interactive Typewriter Hero**: Animates dynamically across key roles: *Python Developer, UI UX Designer, AI Engineer, Web Developer, Freelancer, and Digital Marketer*.
* ⏳ **Narrative & Timeline (About)**: A visual storytelling timeline showcasing academic and professional milestones.
* 🔮 **Skills & Tech Globe**: A modern grid presenting core competencies and tools.
* 💼 **Interactive Projects Grid**: Displays projects with interactive modal-based case studies.
* 🛠️ **Custom Services Grid**: Clearly outlines key solutions and service offerings.
* ⭐ **Star Testimonials**: An elegant testimonial slider displaying verified reviews and endorsements.
* 🖼️ **Graphic & Credential Gallery**: Masonry gallery for certificates, designs, and achievements.
* ✉️ **Spam-protected Contact Form**: Integrated with custom API email routing, frontend validation, a honey-pot anti-spam trap, and a celebration confetti animation.
* 🔒 **SEO Optimized**: Standard SEO setups including `sitemap.ts`, `robots.ts`, and metadata structure.

---

## 🛠️ Tech Stack & Technologies

The application is engineered using cutting-edge frontend tools:

* ⚡ **Core Framework**: [Next.js](https://nextjs.org/) (App Router Architecture)
* ⚛️ **UI Library**: [React 19](https://react.dev/)
* 🎨 **Styling Engine**: [Tailwind CSS v4](https://tailwindcss.com/)
* ✨ **Animations**: [Framer Motion](https://www.framer.com/motion/) & [GSAP (GreenSock)](https://greensock.com/gsap/)
* 📜 **Kinetic Scroll**: [Lenis Scroll](https://lenis.darkroom.engineering/)
* 📁 **Markdown Parser**: [markdown-to-jsx](https://github.com/quantizor/markdown-to-jsx)
* 🎈 **Effects**: [canvas-confetti](https://github.com/catdad/canvas-confetti)
* 💻 **Language**: [TypeScript](https://www.typescriptlang.org/)

## 🚀 Getting Started & Installation

Follow these steps to run the project locally on your machine:

### 1️⃣ Prerequisites
Make sure you have Node.js installed on your machine.
* **Required Node.js Version**: `Node.js >= 20.9.0` (Recommended: v20 or v22 LTS)

### 2️⃣ Clone and Navigate to Project
```bash
cd my_blog_website
```

### 3️⃣ Install Dependencies
Install all the package dependencies:
```bash
npm install
```

### 4️⃣ Run the Development Server
Launch the live local development server:
```bash
npm run dev
```
* Your site will be running live at: **[http://localhost:3000](http://localhost:3000)**

### 5️⃣ Build for Production
To build the application for optimal performance in production:
```bash
npm run build
npm run start
```
